Welcome Admin!

How to:

1. Create user. Fill in username, password and email fields.

2. Fully update or create new user. This is a PUT request, so all fields are required.

3. Update user info. Username, about user, full name and email fields

4. Delete user. Just username.

Then click appropriate action button bellow the form.

If there are any verification errors, server response will be represented in a table bellow the control panel.

SpUserId: CreatedAt: Username: Password: Email:
About User: Avatar Image Url: SpRoleId: Role: Enabled:
Full Name: Account Non Expired: Account Non Locked: Credentials Non Expired
